# Local Agent Instructions

## Technical Plans and Specs

Every technical plan or implementation spec created for this project must be saved in:

```text
docs/specs/
```

Each spec must include both:

- A clear title for the plan.
- A time-date-stamp showing when the plan was created or last materially updated.

Use this header format at the top of each spec:

```markdown
# <Plan Title>

Time-date-stamp: YYYY-MM-DD HH:MM <timezone>
```

Use descriptive, lowercase filenames with hyphens, for example:

```text
docs/specs/structured-profile-generator.md
```
